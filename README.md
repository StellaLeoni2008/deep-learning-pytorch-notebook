## 📌 Sobre

Este repositório reúne os notebooks desenvolvidos ao longo da trilha **Deep Learning com PyTorch**, da Alura.
Durante os estudos, foram explorados os fundamentos das Redes Neurais Artificiais e do Deep Learning utilizando PyTorch, um dos frameworks mais utilizados atualmente para construção e treinamento de modelos de Inteligência Artificial.

---

## 📚 Estrutura da trilha

### 📁 01. Introduction Neural Networks
Introdução aos conceitos fundamentais de Redes Neurais Artificiais utilizando PyTorch.

Neste notebook são explorados os principais componentes de uma rede neural, incluindo tensores, perceptrons, camadas lineares e os conceitos básicos que permitem que modelos neurais aprendam padrões a partir dos dados.

---

### 📁 02. Training Neural Networks
Foco no processo de treinamento de modelos neurais.

Nesta etapa são trabalhados conceitos relacionados a funções de perda, algoritmos de otimização, gradientes, backpropagation, validação e avaliação de desempenho durante o treinamento.

---

### 📁 03. Recurrent Neural Networks
Introdução às Redes Neurais Recorrentes (RNNs).

Os notebooks desta etapa exploram arquiteturas voltadas para processamento de dados sequenciais, permitindo compreender aplicações relacionadas a texto, linguagem natural e séries temporais.

---

### 📁 04. Convolutional Neural Networks
Estudo das Redes Neurais Convolucionais (CNNs).

Nesta etapa são abordados conceitos fundamentais de visão computacional, incluindo convoluções, pooling, extração de características e classificação de imagens utilizando PyTorch.
